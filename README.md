# Como correr el proyecto localmente
Tengo que estar parado en la carpeta juegoContador, luego ejecutar el siguiente comando:
(Da igual si es una CMD, o una bash, funciona igual)

```bash
npm install
```

y finalmente 

```bash
npm run dev
```
